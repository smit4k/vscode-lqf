# vscode-lqf

## Features

This Visual Studio Code extension provides syntax highlighting support for `.lqf` (Lightweight Quick Format) files

## Example `.lqf` file
```lqf
# This is a comment
# LQF format made by smit4k

> server
host >> "localhost"
port >> 8080
use_ssl >> true

> database
engine >> "postgres"
host >> "db.internal"
port >> 5432
username >> "admin"
password >> "s3cr3t"
max_connections >> 20

> features
enabled >> ["search", "analytics", "dark_mode"]
beta >> false
```
